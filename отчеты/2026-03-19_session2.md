## Честная оценка прогресса
**for усвоен хорошо.** Синтаксис for + range() понял быстро, вложенность for + if тоже далась без проблем.

**while — пока слабо.** Концепция "переменная-счётчик/аккумулятор, которую нужно создать заранее с нулём" — не интуитивна. Несколько раз был серьёзный ступор. К концу сессии стало лучше (`session2/lesson2_while3` написал сам по инструкции), но уверенности нет.

## Где были сложности
- **while в целом** — концепция "повторяй пока условие верно" не сразу зашла. Потребовалось 3 объяснения разными способами
- **Переменная-аккумулятор (total = 0)** — не понимал, зачем нужна стартовая точка. Объяснение через "пустой калькулятор" и "ящик" помогло
- **Направление неравенства** — написал `count < 0` вместо `count > 0`
- **Путал print и присваивание** — писал `print(input(...))` вместо `variable = input(...)`
- **Задания были описаны недостаточно подробно** — формулировка "прибавляй к сумме" не говорит новичку "создай переменную = 0". Нужно расписывать каждый шаг явно
- **Кодировка Windows** — русский текст в сравнении строк (while word != "питон") не работал корректно. Добавили .strip()
