# Шпаргалки по сессиям

Тезисы разнесены по файлам: **одна сессия — один файл** в папке [`шпаргалка/`](шпаргалка/).

| Сессия | Файл |
|--------|------|
| 1 — база | [`шпаргалка/сессия_1.md`](шпаргалка/сессия_1.md) |
| 2 — циклы | [`шпаргалка/сессия_2.md`](шпаргалка/сессия_2.md) |
| 3 — списки (в работе) | [`шпаргалка/сессия_3.md`](шпаргалка/сессия_3.md) |

Перед занятием открой нужную сессию или пробеги по порядку только актуальные. Детали уроков — в `sessionN/` и в `отчеты/`.

**Сквозные мини-проекты (корень проекта):** `challenge_sessions_1_2.py` (сессии 1–2), `challenge_sessions_1_3.py` (1–3, после закрытия списков).

**Новые сессии:** добавляй файл `шпаргалка/сессия_N.md` и строку в таблицу выше.
