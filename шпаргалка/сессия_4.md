# Шпаргалка — сессия 4 (словари, кортежи)

| Что | Суть |
|-----|------|
| **dict** | Пары «ключ → значение»; `{ "чай": 100 }` |
| **Доступ** | `словарь["ключ"]` — взять значение |
| **Замена** | `словарь["ключ"] = новое` |
| **Новая пара** | `словарь["новый"] = значение` |
| **for по dict** | `for k in d:` — `k` это ключ; значение: `d[k]` |
| **tuple** | `(a, b)` — порядок есть; без «перезаписи» элемента как в списке |
| **continue** | В цикле: пропустить остаток этой итерации, следующий круг (не break) |

**Уроки:** `lesson4_dict_intro.py` → `lesson4_dict_for.py` → `lesson4_tuple_intro.py` → `lesson4_continue_intro.py` (continue / break); разминки — `lesson4_warmup_*.py`.

**Челлендж (корень проекта):** `challenge_sessions_1_4.py` — столовая: кортеж меню, словарь цен, список заказа, сумма.

**Разминка (списки):** `session4/lesson4_warmup_lists.py` — цикл, append, пустой список, нумерация.
