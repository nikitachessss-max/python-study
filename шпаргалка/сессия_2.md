# Шпаргалка — сессия 2 (циклы)

Кратко, без воды. Детали — в `отчеты/` и в `session2/lesson*.py`.

| Что | Суть |
|-----|------|
| **for** | Повторить **известное** число раз. `for x in range(1, 11):` — числа 1…10 (верхняя граница **не входит**) |
| **range** | `range(3)` → 0,1,2 (три раза). `range(5, 16)` → от 5 до 15 |
| **for + if** | Внутри цикла — `if`, отступ **внутри** for (например чётные: `x % 2 == 0`) |
| **%** | Остаток от деления: `10 % 3 == 1` |
| **while** | Пока условие **истинно** — повторять тело. Сначала часто `input` или счётчик, потом `while ...:` |
| **!=** в while | `while answer != 7:` — пока не угадал |
| **Аккумулятор** | Сумма: `total = 0` до цикла, внутри `total = total + число` |
| **and в while** | `while x != pin and attempts < 3:` — оба условия должны выполняться, чтобы цикл продолжался |

**Важно:** не путать `print(...)` и `переменная = input(...)` — спросить и **сохранить** — это присваивание.

**Вложенность:** `for` снаружи, `while` внутри (или наоборот) — уровни отступов разные (4 / 8 пробелов).

**Строки:** для сравнения с русским текстом у `input()` полезно `.strip()` — убрать пробелы по краям.

**Файлы:** в корне — `warmup2.py`, **`challenge_sessions_1_2.py`** (итоговая задача сессий 1+2). В `session2/`: `lesson2_for.py`, `lesson2_for2.py`, `lesson2_while.py`, `lesson2_while2.py`, `lesson2_while3.py`, `lesson2_final.py`, `lesson2_pin.py`, `lesson2_extra.py`, `lesson2_extra2.py`
