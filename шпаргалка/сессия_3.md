# Шпаргалка — сессия 3 (списки, в работе)

Кратко, без воды. Детали — в `отчеты/` и в `session3/lesson*.py`.

| Что | Суть |
|-----|------|
| **list** | Список: `[a, b, c]` — несколько значений в одной переменной, порядок важен |
| **Индекс** | Номер элемента **с нуля**: первый — `[0]`, второй — `[1]`, третий — `[2]` |
| **Доступ** | `имя[индекс]` — взять один элемент |
| **Замена** | `имя[индекс] = новое` — элемент на позиции меняется |
| **len** | `len(список)` — сколько элементов; последний индекс: `len - 1` |
| **Срез** | `список[a:b]` — кусок списка; **b не входит**. Примеры: `[:3]`, `[2:]` |
| **for по списку** | `for x in список:` — по одному элементу, без индекса |
| **range vs список** | `range` → счётчик 0, 1, 2…; список → твои значения по очереди |
| **append** | `список.append(x)` — дописать x в конец; список сам меняется |
