# Шпаргалка — сессия 1 (база)

Кратко, без воды. Детали — в `отчеты/` и в `session1/lesson*.py`.

| Что | Суть |
|-----|------|
| **Переменные** | `имя = значение`. Типы: `int` (целое), `float` (дробь), `str` (текст в кавычках) |
| **print** | Вывод на экран. **f-строка:** `print(f"текст {переменная}")` — в `{}` подставляется значение |
| **input** | Ввод с клавиатуры — всегда **строка**. Число: `int(input(...))` или `float(input(...))` |
| **if / elif / else** | Условие, потом `:`, тело с **отступом**. `elif` — иначе если. `else` — **без** условия, «все остальные» |
| **Сравнения** | `==` равно, `!=` не равно, `<` `>` `<=` `>=` |
| **and** | И то и другое: `age >= 18 and age <= 64` |

**Памятка:** Python идёт сверху вниз; переменные, созданные выше, видны ниже. Сначала сохраняешь в переменную (`discount = ...`), потом используешь в формуле.

**Файлы:** `session1/lesson1.py`, `session1/lesson1_input.py`, `session1/lesson1_if.py`, `session1/lesson1_final.py`
